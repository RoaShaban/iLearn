# iLearn

A mock e-learning platform built as a multi-page website — my first multi-page front-end project. Built to practice HTML, CSS, and Bootstrap in a structured, real-world-like project.

## 🎯 Purpose

This project was built to apply and strengthen front-end development skills by designing and structuring a complete multi-page website from scratch.

## 🛠️ Built With

- HTML5
- CSS3
- Bootstrap

## 📄 Pages

- **Home** — landing page introducing the platform
- **Courses** — listing of available courses
- **Single Course** — detailed view of an individual course
- **Benefits** — highlights the advantages of using iLearn
- **Pricing** — available subscription plans and pricing tiers

## ✨ What I Learned

- How to structure and link multiple HTML pages together
- How to build consistent layouts across pages using Bootstrap
- How to design a clean and user-friendly interface
- How to organize a front-end project with multiple files and assets
